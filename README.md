# JLPT Grammar Graph Database

This is not a traditional Anki deck. It is a **relational grammar database** designed to mirror how the human brain actually stores information: through connections, not lists. 

Instead of treating grammar points as isolated objects, this system visualizes the relationships between them as an **interconnected "Family Tree" (Erbbaum)** with adjustable degrees of relation.

## ⚠️ Core Requirement

To use this deck as intended (navigating between related cards and maintaining the link structure), you **must** install the following Anki add-on:

* **Link Cards Add-on:** [AnkiWeb ID 1077002392](https://ankiweb.net/shared/info/1077002392)

---

## 🧠 The Concept: Grammar as a Network

Traditional learning is often siloed by JLPT levels. This database breaks those walls. A card is considered "related" based on three main pillars:
1. **JLPT Level Proximity:** How close they are in the formal testing hierarchy.
2. **Phonetic Similarity:** "How similar they sound" (e.g., structures sharing the same keywords like *wake*, *mono*, or *nagara*).
3. **Semantic Overlap:** How closely the meanings or nuances align.

### The "Family Tree" (Relation Degree)
You can explore these connections through a visual graph. This allows you to see the "surface" of Japanese grammar while simultaneously digging into the deep-rooted connections.

### 3. The "Effortless" Goal
Grammar is the core of language. Understanding it is one thing; decompressing it in real-time is another. 
* **Speed:** This deck optimizes your reading speed—the ultimate benchmark for passing the JLPT.
* **Side Effects:** Improved listening comprehension is a natural byproduct of mastering these structures. Once grammar becomes effortless, skimming large bodies of text becomes possible.

## 🛠 How to Use
1. **Install the Link Cards Add-on** (link above).
2. **Import the Deck.**
3. **Navigate via Neighbors:** Use the direct links on each card to jump to "neighboring" grammar points.
4. **Compare & Contrast:** Use the overview to understand why two seemingly similar points (e.g., *nagara* vs. *nagara ni*) are different.

## 📝 Philosophy
"Relations are abstract and not always mutual. Every grammar point is a standalone object, but the more connections you understand, the broader your mastery becomes. It’s like climbing a mountain—the air gets thin, but the view (and the ability to use the language) becomes crystal clear."
